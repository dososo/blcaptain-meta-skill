# 视觉制作规格

## 画布与安全区

- 默认画布：竖版 3:4，推荐 1242 x 1660 或 1080 x 1440。
- 安全区：四边至少保留画布短边的 6% 至 8%，标题和头像不要贴边。
- 封面要适合小红书瀑布流缩略图，标题区域应在缩略状态也能读出关键词。
- 每张卡保持同一套网格、字号层级和角标位置，避免六张像六套模板。

## 信息层级

每张卡最多使用四级信息：

1. 主标题：唯一最醒目的句子。
2. 解释正文：1 至 3 行短句或 3 个 bullet。
3. 证据标注：箭头、圈注、局部放大、截图标签。
4. 页码或栏目标签：小而稳定，不抢正文。

不要把所有文字放进圆角矩形标签。按钮、标签、气泡、便签只在表达关系时使用。

## 视觉方向

优先选择能服务内容的视觉，而不是套固定风格：

- 知识观点：清爽编辑感、强标题、结构图、少量强调色。
- 产品解析：真实截图为主，配放大框、步骤标注、浅色背景。
- 个人经验：头像或人物照建立信任，配手写式标注或清单。
- 方法教程：流程线、编号卡、检查表、前后对比。

配色建议使用“中性底色 + 深色正文 + 一个主强调色 + 一个辅助信号色”。避免全篇只有一种颜色的深浅变化。

## 图片生成提示词模板

当需要生成卡片底图、封面或插画时，用这个结构写提示词：

```text
竖版 3:4 小红书图文卡片，主题：[主题]，目标读者：[人群]。
画面主体：[头像/产品截图/抽象结构/场景物件]。
构图：[标题位置]，[主体位置]，[标注方式]。
视觉风格：[清爽编辑感/产品解析/个人经验/方法教程]。
文字区域：预留清晰中文标题和短正文区域，不生成乱码文字。
色彩：[中性底色]，[主强调色]，[辅助信号色]。
质感：高清、干净、手机阅读友好、信息层级明确。
```

负面提示词：

```text
乱码中文、错误 UI、虚构产品截图、过密小字、低清截图、强噪点、夸张营销海报、文字遮挡人脸、标题超出画布、无关装饰
```

## 产品截图排版

- 保持截图比例，不要横向压扁或纵向拉伸。
- 用 1 至 3 个标注解释截图，不要把截图变成背景纹理。
- 重点区域可放大 140% 至 220%，但要保留原截图上下文。
- 如果截图来自真实产品，图片提示词里写“保留用户提供的真实截图，不重绘 UI”。
- 如果没有截图，只能写“产品截图占位”，不要生成看似真实的新界面。

## 头像排版

- 头像适合小尺寸信任背书：封面署名、第 6 张作者建议、案例旁注。
- 头像不应遮挡标题，也不应被强行做成大面积背景。
- 人脸周围保留呼吸空间；需要裁切时保留眼睛和嘴部表情。

## 出图质检

交付图片或设计稿前逐张检查：

- 标题是否完整、无错别字、无乱码。
- 手机缩略图下第 1 张是否还能识别核心关键词。
- 正文是否过密；最长一行是否明显超出视觉容器。
- 截图、头像、产品名是否来自用户材料或明确标为占位。
- 六张卡的页码、边距、字体层级是否一致。
- 第 5 张是否真的证明了方法，而不是随手放一张截图。
- 第 6 张是否给出明确行动，不只是“谢谢观看”。
